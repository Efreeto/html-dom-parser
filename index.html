HTML DOM PARSER

 

This program loads another web page to parse its internal structure, then turn it into a single html file.

Its purpose is to help other non-browser, web crawler programs with no ability to parse into the internal structure on their own. (For example, wget with no option or an in-house company tool that can only read in the first page and not its dependencies)

 

Usage:

/?url=url&halttime=halttime&waittime=waittime

url : web address for the raw html

halttime : (optional) interrupt page loading after this time. Used when page loading takes too long. Unit is millisecond.

waittime : (optional) extend page loading at least for this time. Used when page loading finishes before writing any useful data. Unit is millisecond.

 

Examples:

/?url=https://www.youtube.com

It shows the raw html the website without making any recommendations from the youtube account you may have.

 

/?url=http://radiosearchengine.com&halttime=1000

This website is now offline. This site used to take too long to load, but interrupting the loading with 'halttime' still gave useful information.

 

/?url=http://www.biography.com/people/barack-obama-12782369&waittime=5000

This website returns results too quickly and without any useful information if you are not browsing properly. But if the loading time is extended with 'waittime', it returns results correctly.
