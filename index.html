HTML DOM PARSER

 

Usage: /?url=url&halttime=halttime&waittime=waittime

url : web address

halttime : (optional) interrupt page loading after this time. Used when page loading takes too long. Unit is millisecond.

waittime : (optional) extend page loading at least for this time. Used when page loading finishes before writing useful data. Unit is millisecond.

 

Example:

 /?url=https://github.com/

 /?url=http://radiosearchengine.com&halttime=1000

 /?url=http://www.biography.com/people/search/obama&waittime=5000

 

The purpose of this program is to parse out internal structures of a web page into a single html file.

It is designed to help other html reading programs (like wget with no option) that can only read in the first page and not its dependencies.
