HTML DOM PARSER

 

Usage: /?url=url&timeout=timeout

url : web address

timout : (optional) cut off time

 

Example:

 /?url=http://www.biography.com

 /?url=http://radiosearchengine.com&timeout=3000

 /?url=http://www.biography.com/people/search/obama&waittime=5000
